# ビジネスモデル

## 収益モデル（案）

### サブスクリプション（月額/年額）

| プラン | 想定価格帯 | 主な機能 | ターゲット |
|--------|------------|----------|------------|
| **Free** | 無料 | 月○投稿まで、基本機能のみ | トライアル・個人の軽い運用 |
| **Starter** | 月額 980〜1,980円 | 月○投稿、文章生成・画像補正 | 個人クリエイター |
| **Pro** | 月額 3,980〜5,980円 | 無制限に近い投稿、複数アカウント | 中小企業・本格運用者 |
| **Business** | 月額 9,800〜19,800円 | チーム機能、優先サポート、API | 代理店・企業 |

※価格はあくまで想定。市場調査・コスト分析に基づき決定する。

### その他の収益源（検討）

- **従量課金**: 文章生成・画像処理の利用量に応じた課金（高用量ユーザー向け）
- **エンタープライズ契約**: 大企業向けのカスタマイズ・オンプレミス対応
- **アフィリエイト・提携**: 関連サービス（フォトストック、AIサービス等）との連携による紹介料

## コスト構造（想定）

| 項目 | 内容 |
|------|------|
| **インフラ** | クラウド（AWS/GCP/Azure）、AI API利用料、ストレージ |
| **SNS API** | 各プラットフォームのAPI利用（無料枠内での運用を基本とする） |
| **開発・運用** | エンジニア、デザイナー、カスタマーサポート |
| **マーケティング** | 広告、コンテンツマーケティング、パートナーシップ |

---

## 主要指標（KPI案）

| 指標 | 説明 | 目標例 |
|------|------|--------|
| **MAU** | 月間アクティブユーザー数 | 初年度 ○○人 |
| **有料転換率** | Free → 有料プランへの転換率 | ○% |
| **解約率（Churn）** | 月次解約率 | ○%以下 |
| **LTV** | 顧客生涯価値 | 単価 × 平均継続月数 |
| **CAC** | 顧客獲得コスト | LTVの○分の1以下 |

## 成長戦略（案）

### Phase 1: ローンチ・検証（0〜6ヶ月）
- MVPのリリース
- 初期ユーザー獲得（βテスター、口コミ）
- プロダクトマーケットフィットの検証

### Phase 2: 成長（6〜18ヶ月）
- 有料プランの本格展開
- コンテンツマーケティング・SNS広告による認知拡大
- 機能拡充（動画対応、アナリティクス強化）

### Phase 3: スケール（18ヶ月〜）
- エンタープライズ向け展開
- 海外市場の検討
- パートナー・チャネル拡大

---

## 投資・資金調達（検討事項）

- **ブートストラップ**: 自己資金・売上で開発・運用
- **シード/プレシード**: エンジェル、VCからの資金調達
- **補助金・助成金**: 国・自治体の起業支援制度の活用
